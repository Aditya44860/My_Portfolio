# 🌌 Aditya Bhardwaj — Developer Portfolio

A premium, cinematic developer portfolio built with **React**, **Tailwind CSS**, and **Framer Motion**. This project showcases a "Cyber-Studio" aesthetic, featuring immersive micro-animations, liquid glass effects, and dynamic data integration.

> ✨ *Building high-performance digital experiences at the intersection of Software Engineering and Data Science.*

---

## 🚀 Core Features

- ✦ **Dynamic GitHub Integration**: Automatically synchronizes with the GitHub API to fetch pinned repositories, real-time stars, and metadata.
- ✦ **Cinematic Animations**: Implements advanced scroll-driven animations and fluid transitions using Framer Motion.
- ✦ **Liquid Glass UI**: A high-end dark mode interface featuring sophisticated glassmorphism and vibrant neon accents.
- ✦ **Interactive Technologies**: A responsive grid of animated icons showcasing a comprehensive tech stack.
- ✦ **Project Fallbacks**: Engineered resilience with hardcoded fallbacks to ensure a premium experience even during API outages.

---

## 🛠 Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Motion**: Framer Motion
- **Icons**: React Icons, Lucide React
- **Data**: GitHub REST API
- **Deployment**: Netlify / Vercel

---

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aditya44860/My_Portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
